# Raycast Integration

The Brag Raycast extension provides a seamless way to track accomplishments without leaving your keyboard.

## Commands

### Add Brag
Quickly capture an accomplishment.
- **Content**: What you did (supports Markdown)
- **Project**: Associated project (optional)
- **Tags**: Comma-separated tags (optional)

### Search Brags
Full-text search through your history.
- **Enter**: Copy content to clipboard
- **Cmd+Enter**: Copy as JSON

### Recent Brags
View accomplishments from the last 7, 14, 30, or 90 days.
- Perfect for weekly updates or standups.

### Export Brags
Export data for a specific date range.
- Formats: JSON, Markdown, Plain Text
- Output is automatically copied to clipboard.

## Troubleshooting

If commands fail, ensure `brag` is in your PATH. The extension attempts to execute `brag` directly.
